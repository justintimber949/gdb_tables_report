Methods & Abbreviations

How to Read the Values

📋 All Number and ASR values follow format: value (lower to upper) — consistent with international GBD journal reporting standards.
ℹ
Value format (GBD standard):
Number: 4,656.29 (1,015.20 to 8,315.82) → point estimate (95% UI lower to upper)
ASR: 4.4316 (0.9662 to 7.9144) per 100,000 population
EAPC: 0.11 (0.01 to 0.20) → point estimate (95% CI lower to upper)
UI
95% Uncertainty Interval (UI) — Number & ASR

Values in parentheses for Number and ASR columns are 95% Uncertainty Intervals from the GBD model (computed from 1,000 draws/simulations).

Unlike standard statistical CI — UI reflects uncertainty across the full GBD estimation pipeline, including data quality, model assumptions, and sampling variation.

Example: 4,656 (1,015 to 8,316) means point estimate = 4,656 deaths, 95% range 1,015 to 8,316.

CI
95% Confidence Interval (CI) — EAPC

Parenthetical values in the EAPC column are 95% CIs from log-linear regression of ln(ASR) ~ year over 1990–2023.

CI lower > 0Significant increase — entire CI above zero
CI upper < 0Significant decrease — entire CI below zero
CI crosses 0Non-significant trend (CI spans zero)
—Not estimable (insufficient positive rate values)
F
EAPC Formula
# Step 1: OLS regression
ln(ASR_t) = α + β × year

# Step 2: Convert to %
EAPC = (e^β − 1) × 100

# 95% CI
lower = (e^(β−1.96·SE) − 1) × 100
upper = (e^(β+1.96·SE) − 1) × 100
V
Special Value Codes
(empty)Section header row — visual separator, not data
N/A (not linked)Risk factor has no established causal link to this cancer in GBD model
N/A (data absent)High LDL — not available in this GBD 2023 dataset
negative valueFrom GBD data — not an error. Interpret as "burden near zero / not reliably detected." Write as 0 or — in publication.
A
Two Types of ASR in Tables

Age group rows → age-specific rate for one age stratum only (Global). Not ASMR. Smaller values at younger ages reflect low cancer burden.

Location & Cancer type rows → true ASMR/ASDR, computed using WHO Standard Population 2000–2025 weights across all age groups. Use these for cross-location comparisons.

!
Negative Values at Young Ages
⚠
Not an error. These come directly from GBD model output.

Bayesian GBD model (DisMod-MR) for rare cancers in young age groups (20–49) can produce point estimates and UIs that include negative values.

Substantively: "burden approaching zero / not reliably detected." In manuscript, report as 0 or — with footnote: "Negative values represent estimates within the GBD uncertainty range and should be interpreted as negligible attributable burden."

Methods & Data Sources

Data Source

Global Burden of Disease (GBD) 2023. Institute for Health Metrics and Evaluation (IHME), University of Washington. Accessed via GBD Results Tool.

Cancers Analyzed

Breast cancer (cause=429), Cervical cancer (432), Uterine cancer (435), Ovarian cancer (465) — GBD cause codes.

Risk Factors

High BMI (rei=108, ≥25 kg/m²), Smoking (rei=99), High alcohol use (102), High fasting plasma glucose (105), High LDL (data absent), Diet red meat (116), Low physical activity (125), Unsafe sex (170).

Age Standardization

WHO World Standard Population (2000–2025). 16 age groups: 20–24 to 95+. ASMR/ASDR computed by weighted sum of age-specific rates using WHO weights.

EAPC Calculation

Log-linear regression of age-standardized rates on calendar year. EAPC = (e^β−1)×100. 95% CI from slope±1.96×SE. Requires ≥3 non-zero rate values.

Geographic Data

Natural Earth 10m admin-0 country boundaries. GBD location_id → ISO-3166 alpha-3 mapping. 204 countries and territories. Engine: geopandas + pyogrio.

Choropleth Bins

% change panels: <0% (decrease), 0–100%, 100–300%, 300–500%, 500–1000%, >1000%. Continuous ASMR/ASDR panels: YlOrRd and PuRd colormaps.

Input Files

gbd_global.csv, gbd_indonesia.csv, gbd_lowsdi…highsdi.csv (main), datanegaraupdate.csv (204-country choropleth), TABLE3_OtherRiskFactors.csv.

Software

Python 3.13 — pandas, numpy, matplotlib, geopandas (pyogrio engine), scipy.stats. All figures 300 DPI PNG.
